# 获取sample依赖的资源文件到本地

## 1.获取get_resource.sh脚本

* 具体通过什么方式得到脚本，待更新

## 2.运行脚本获得resource

* 将get_resource.sh脚本文件复制到device/soc/hisilicon/hi3516dv300/sdk_linux/sample/目录下。
* 执行下面的命令，下载资源文件到对应的目录下。

```
chmod 777 get_resource.sh

./get_resource.sh
```
